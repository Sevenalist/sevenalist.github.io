    Mkdocs撰写技巧¶

    1 提示框¶

    1.1普通提示框¶

    普通提示框

    这是一个提示框案例。

    首先需要在mkdocs.yaml文件中添加以下参数：

    YAML
    markdown_extensions:
  - admonition              # 提示框功能
  - pymdownx.details        # 提示框折叠功能
  - pymdownx.superfences    # 提示框中添加代码
    使用方法为：

    Markdown
    !!! note "普通提示框"
    这是一个提示框案例。

    1.2 折叠提示框¶

    折叠提示框

    这是一个提示框案例。

    如果想要创建折叠提示框，只需要将开头的感叹号!替换为问号?即可。

    实现代码如下：

    Markdown
    ??? note "折叠提示框"
    这是一个提示框案例。

    除了note类型的提示框，官网还提供了诸多其他种类，详情可查阅官方文档。
